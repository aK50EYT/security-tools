# Security Tools 🔒

A collection of offensive security tools for bug bounty and penetration testing.

## Tools

### Port Scanner
A Python-based port scanner with the following features:
- Scan multiple ports
- Domain name resolution
- Service detection (HTTP, SSH, FTP...)

## Usage
```bash
cd port-scanner
python scanner.py
```

## Example Output
```
[*] google.com resolved to 172.217.171.46
[*] Scanning google.com...
[+] Port 80 OPEN  →  HTTP
[+] Port 443 OPEN →  HTTPS
[-] Port 22 CLOSED
```
